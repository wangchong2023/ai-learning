# Multi-Agent Lab: 多智能体协作实验场

本模块展示了 LangGraph 最核心的进阶能力：**多智能体接力协作 (Agent Relay)**。

## 🧩 为什么需要多智能体？
在处理复杂任务（如撰写深度技术分析）时，单个 Agent 往往难以同时兼顾“海量资料检索”与“文字精雕细琢”。通过分工：
1.  **Researcher (研究员)**：专注挖掘事实，不考虑措辞。
2.  **Writer (作家)**：专注结构与修辞，基于研究员提供的“事实包”进行创作。

## 🔄 核心流转逻辑
1.  **输入**：用户提出一个复杂命题。
2.  **研究阶段**：研究员 Agent 被激活，它会将搜集到的事实存入状态机的 `research_notes` 字段。
3.  **接力**：状态机根据 `next_agent` 路由到作家。
4.  **创作阶段**：作家 Agent 读取 `research_notes`，转化为最终文章。

## 🚀 启动
```bash
make multi
```
